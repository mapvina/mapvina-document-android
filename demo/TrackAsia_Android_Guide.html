Tài liệu dành cho Tech Lead/PM/Senior Dev. Mục tiêu: tích hợp nhanh TrackAsia Map (Map + Navigation) vào ứng dụng Android, chuẩn hoá cấu hình, vòng đời, và best practices.

1) Yêu cầu môi trường

  • Android Studio mới (AGP 8.4.x), Kotlin 1.9.x
  • compileSdk 34, targetSdk 35, minSdk 26

2) Cài đặt SDK & Gradle

Thêm repositories trong dự án (google(), mavenCentral()). Thêm dependencies TrackAsia trong module app:

// app/build.gradle (trích)
dependencies {
  implementation 'io.github.track-asia:android-sdk:2.0.2'
  implementation 'io.github.track-asia:geojson:2.0.2'
  implementation 'io.github.track-asia:turf:2.0.2'
  implementation 'io.github.track-asia:android-plugin-annotation-v9:2.0.1'
  implementation 'io.github.track-asia:navigation-core:2.0.2'
  implementation 'io.github.track-asia:navigation-ui-android:2.0.2'
}
Khuyến nghị quản lý API key bằng Secrets Gradle Plugin, không hardcode key trong mã nguồn.

3) Quyền & AndroidManifest

  • Bắt buộc: INTERNET, ACCESS_NETWORK_STATE
  • Nếu dùng vị trí: ACCESS_FINE_LOCATION, ACCESS_COARSE_LOCATION
  • Nếu dùng UI điều hướng: khai báo Activity navigation của TrackAsia

4) Thêm MapView vào layout XML

<com.trackasia.android.maps.MapView
    android:id="@+id/mapView"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    app:trackasia_cameraZoom="12"
    app:trackasia_uiCompass="true"
    app:trackasia_uiZoomGestures="true" />

5) Khởi tạo map trong Fragment/Activity

// Kotlin (tối giản)
private lateinit var trackasiaMap: com.trackasia.android.maps.TrackAsiaMap
private var styleUrl = "https://maps.track-asia.com/styles/v1/streets.json?key=public"

override fun onViewCreated(view: View, savedInstanceState: Bundle?) {
  com.trackasia.android.TrackAsia.getInstance(requireActivity())
  val mapView = view.findViewById<com.trackasia.android.maps.MapView>(R.id.mapView)
  mapView.onCreate(savedInstanceState)
  mapView.getMapAsync { map ->
    trackasiaMap = map
    map.setStyle(com.trackasia.android.maps.Style.Builder().fromUri(styleUrl)) { style ->
      // TODO: thêm marker, route...
    }
  }
}

6) Quản lý vòng đời MapView

  • onStart → mapView.onStart()
  • onResume → mapView.onResume()
  • onPause → mapView.onPause()
  • onStop → mapView.onStop()
  • onDestroyView/Destroy → mapView.onDestroy()
  • onLowMemory → mapView.onLowMemory()
  • onSaveInstanceState → mapView.onSaveInstanceState(outState)

7) Quyền vị trí & vị trí người dùng

Xin quyền runtime bằng PermissionsManager. Lấy vị trí hiện tại qua trackasiaMap.locationComponent.lastKnownLocation.

8) Marker, Camera, Style

// Thêm marker (minh hoạ)
val iconFactory = com.trackasia.android.annotations.IconFactory.getInstance(context)
val marker = com.trackasia.android.annotations.MarkerOptions()
  .position(com.trackasia.android.geometry.LatLng(21.028, 105.854))
trackasiaMap.addMarker(marker)

// Camera
val update = com.trackasia.android.camera.CameraUpdateFactory.newLatLngZoom(
  com.trackasia.android.geometry.LatLng(21.028, 105.854), 14.0)
trackasiaMap.animateCamera(update, 1000)

// Đổi style runtime
trackasiaMap.setStyle("https://maps.track-asia.com/styles/v1/night.json?key=public")

9) Routing & Navigation

Tính route bằng Navigation Core, parse DirectionsResponse; vẽ route bằng NavigationMapRoute. Có thể chuyển sang TrackAsiaNavigationActivity để điều hướng turn-by-turn.

11) Checklist tích hợp nhanh

  • Thêm dependencies TrackAsia; đồng bộ version.
  • Cấu hình quyền/Activity trong Manifest.
  • Thêm MapView vào layout, khởi tạo SDK, set style URL.
  • Forward đủ lifecycle MapView.
  • Kiểm thử marker/camera/navigation + xin quyền vị trí.
  • Cấu hình ABI filters, logging và proguard khi cần.
